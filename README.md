# KP-Magang-Jogja

## Prerequisites
- [composer](https://getcomposer.org/)
- [NodeJS](https://nodejs.org/en/)
- [NPM](https://nodejs.org/en/) or [Yarn](https://yarnpkg.com/)

## Getting started
1. install composer packages
```bash
composer install
```
2. install NPM packages
- with NPM
```bash
npm install
```
- with Yarn
```bash
yarn install
```
3. Run the application
```bash
php artisan serve
```

4. Develop
```bash
yarn watch
```

5. Production
```bash
yarn prod
```
